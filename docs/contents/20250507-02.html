1. Golay符号の基本概念

Golay符号とは

Golay符号 [Gol49] は、完全符号の一つとして知られている重要な線形符号です。

Golay符号は、Marcel J. E. Golayによって1949年に開発された線形符号で、Hamming符号と同様に完全符号であり、強力な誤り訂正能力を持ちます。

Golay符号の特徴

  • 主に2元(23, 12, 7)符号と3元(11, 6, 5)符号の2種類がある
  • 完全符号(perfect code)である
  • 2元(23, 12, 7) Golay符号は最大3重誤り訂正が可能

完全符号 (perfect code)：

重み $t$ 以下のすべてのパターンをコセットリーダとし、それ以外のパターンをコセットリーダとしてもたない符号。

Hamming符号とGolay符号は、重要な完全符号の例です。

2. 2元(23, 12, 7) Golay符号

2元(23, 12, 7) Golay符号の定義

2元(23, 12, 7) Golay符号が完全符号であることは、以下の等式が成り立つことから確認できます：

$2^{12} \left[\binom{23}{0} + \binom{23}{1} + \binom{23}{2} + \binom{23}{3}\right] = 2^{23}$

この等式は、(23, 12, 7) Golay符号が完全符号であることを示しています。つまり、誤り訂正能力が最適であり、すべての可能なシンドロームに対して明確な対応関係があります。

パリティ検査行列

以下に2元(23, 12, 7) Golay符号 $G_{23}$ のパリティ検査行列 $P^T$ の例を示します：

P^T = [ 100111000111 101011011001 101101101010 101110110100 110011101100 110101110001 110110011010 111001010110 111010100011 111100001101 011111111111 ]

定理 2.3.1：

2元(23, 12, 7) Golay符号 $G_{23}$ は完全符号である。

3. 拡大Golay符号

拡大Golay符号の定義と特性

定理2.2.2（Hamming符号のスライドで示したもの）より、2元(23, 12, 7) Golay符号 $G_{23}$ から拡大Golay符号を構築できます。

定理 2.3.2：

拡大Golay符号 $G_{24}$ は自己双対符号、すなわち $G_{24}^{\perp} = G_{24}$ である。

拡大Golay符号の特性

  • 2元(24, 12, 8)符号として定義される
  • 符号語の重みはすべて4の倍数
  • 自己双対符号である（$G_{24}^{\perp} = G_{24}$）
  • 任意の位置の1つの記号を除去すると元のGolay符号 $G_{23}$ が得られる

拡大Golay符号 $G_{24}$ は、符号理論において非常に興味深い性質を持つ重要な符号の一つです。自己双対という特性は特に重要で、これは符号とその双対符号が一致することを意味します。

誤り訂正能力

拡大Golay符号 $G_{24}$ は、最小距離 $d = 8$ であるため、3重誤りまで訂正でき、4重誤りまで検出できます。

4. Golayとその他の完全符号の関係

完全符号の特徴と制限

その他の完全符号

  • 3元(11, 6, 5) Golay符号も完全符号である
  • 注目すべき点として、次式が成り立つにもかかわらず、2元(90, 78, 5)符号は存在しないことが証明されている：
$2^{78} \left[\binom{90}{0} + \binom{90}{1} + \binom{90}{2}\right] = 2^{90}$

定理：

自明でない $q$ 元完全符号は、Hamming符号またはGolay符号と同一のパラメータを持つ。ここで、$q$ は素数のべき乗である。

完全符号は、その優れた性質により理論的に非常に重要ですが、Hamming符号とGolay符号（および自明な繰り返し符号）以外の完全符号は存在しないことが知られています。これは符号理論における重要な結果です。

まとめ

  • Golay符号は、符号理論において極めて重要な完全符号の一つ
  • 2元(23, 12, 7) Golay符号は3重誤りまでの訂正が可能
  • 拡大Golay符号(24, 12, 8)は自己双対符号であり、独特の数学的性質を持つ
  • 3元Golay符号(11, 6, 5)も完全符号として知られている
  • 自明でない完全符号は、Hamming符号またはGolay符号と同一のパラメータを持つ

Golay符号は、その完全符号としての性質と優れた誤り訂正能力から、通信システムや情報理論において重要な役割を果たしています。拡大Golay符号の自己双対性という特性は、符号理論と代数的組み合わせ論の美しい関係を示しています。
